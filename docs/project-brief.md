# Project Brief — Photography Portfolio

## Goal

Create a personal photography portfolio website for Zakaria Choukri.

The website should present selected photography work in a clean, elegant, and professional way.

The site will be hosted on GitHub Pages at:

https://zakariaCHOUKRI.github.io/photography/

## Primary Audience

The site is intended for:

- people interested in Zakaria's photography
- potential collaborators
- friends and personal contacts
- creative communities
- potential clients, if the portfolio later becomes professional

## Main Objectives

The website should:

1. Showcase photography beautifully.
2. Load quickly.
3. Work well on mobile and desktop.
4. Be easy to update with new galleries.
5. Avoid unnecessary complexity.
6. Feel personal, minimal, and polished.

## Desired Pages

The website should include:

- Home
- Galleries
- Individual gallery pages
- About
- Contact

## Visual Direction

The website should feel:

- minimal
- artistic
- calm
- modern
- spacious
- image-first

The design should not distract from the photos.

## Content Priorities

The most important content is the photography itself.

Text should be minimal and supportive.

Suggested homepage sections:

1. Hero image
2. Short introduction
3. Featured galleries
4. Selected photographs
5. Link to contact or Instagram

## Constraints

The site must be:

- fully static
- hosted on GitHub Pages
- built with Astro
- deployed with GitHub Actions
- easy to maintain from VS Code
- compatible with the `/photography/` base path

No backend should be used.
